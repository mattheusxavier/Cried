# Cried - Seu encurtador de URL
A simple micro-Saas for URL shortening.

"Obs. O intuito principal do projeto erá o deploy na plataforma da amazon(AWS), mas devido a problemas de acesso até a presenta data, impossibilitando o deploy na plataforma, entretanto foi utilizado outros serviços:
Heroku - BackEnd
Netlify - FrontEnd"

APP - https://cried.netlify.app/

API - https://cried.herokuapp.com/
